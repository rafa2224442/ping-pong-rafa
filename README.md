# ping-pong-rafa

#Deskripsi
Proyek ini adalah game ping - pong sederhana berbasis Python menggunakan pustaka Pygame. Game ini dibuatsebagai latihan pemograman python untuk memahami konsep sprite, event-handling dan logika permainan.

Dua pemain dapat bermain di satu komputer. Pemain menggerakan raket untuk memantulkan bola. Pemain yang gagal memantulkan bola akan dinyatakan kalah

# fitur game
-2 Pemain (kontrol keyboard)
-Bola bergerak otomatisdengan pantulan realistis
-Aturan menang - kalah otomatis
-Tampilan sederhana namun sudah dikembangkan

#Kontrol Permainan 
Pemain 1 (raket kiri)
- W = Naik
- S = turun

Pemain 2 (Raket kanan)
- UP Arrow = Naik
- Down Arrow = Turun

# Instalasi dan cara main
1. Pastikan Python9 sudah terinstall
2.Install Pygame:
pip install pygame or python -m pip install pygame
3. Clone repositori ini:
   git clone https://github.com/rafa2224442/ping-pong-rafa/edit/main/README.md
4. Jalankan game:
   python ping_pong.py
